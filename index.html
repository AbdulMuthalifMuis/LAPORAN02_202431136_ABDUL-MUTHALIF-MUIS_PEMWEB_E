a. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web modern?

CSS dan JavaScript digunakan dalam pemrograman web modern untuk memisahkan struktur (HTML), presentasi (CSS), dan fungsionalitas (JavaScript). CSS menangani tampilan visual agar menarik dan responsif, sementara JavaScript menambahkan interaktivitas dan membuat halaman web dinamis. Kombinasi ketiganya menciptakan pengalaman pengguna yang kaya dan lengkap.

b. Jelaskan konsep dan tujuan utama dari Responsive Web Design (RWD)!

Konsep Responsive Web Design (RWD) adalah merancang satu situs web yang dapat beradaptasi tampilannya secara otomatis di berbagai ukuran layar dan perangkat, mulai dari desktop hingga smartphone. Tujuan utamanya adalah menciptakan pengalaman pengguna (UX) yang optimal dan konsisten, dengan menyediakan kemudahan akses, pembacaan, dan navigasi tanpa perlu membuat versi situs yang berbeda untuk setiap perangkat.

c. Bagaimana kombinasi CSS, JavaScript, dan prinsip RWD

CSS, JavaScript, dan prinsip Responsive Web Design (RWD) bekerja bersama untuk menciptakan halaman web yang menarik, interaktif, dan nyaman di semua perangkat. CSS mengatur tampilan visual serta responsivitas dasar melalui layout fleksibel dan media queries, sementara JavaScript menambahkan interaksi seperti animasi, validasi, atau toggle menu pada layar kecil. Prinsip RWD memastikan seluruh elemen — termasuk teks, gambar, dan struktur — dapat menyesuaikan ukuran layar secara otomatis. Ketiganya saling melengkapi: CSS menentukan gaya, JavaScript memberi fungsi dinamis, dan RWD menjadi panduan agar tampilan tetap optimal di mobile, tablet, hingga desktop.
